========================================================================
                      本地 TTS-API 服務啟動指南
========================================================================

【重要前提】
⚠️ 閱讀以下資訊前，建議先確保你擁有了在屬於自己的域名（網域）。
   如沒有則不建議啟動本地的 TTS-API 服務！

------------------------------------------------------------------------
一、 準備工作與下載
------------------------------------------------------------------------
1. 下載 GPT-SoVITS 整合包：
   請先到以下網址找適用你的 GPT-SoVITS 整合包並下載 (必須要支持v4版本或以上)。
   網址：https://www.yuque.com/baicaigongchang1145haoyuangong/ib3g1e/dkxgpiy9zb96hob4#nVNhX
   （作者是 b 站 up 主：花儿不哭 / 來源：https://b23.tv/XNB0DiA）

2. 下載已訓練好的模型：
   網址：https://drive.google.com/drive/folders/1zt8rjSG4K7xtbmUXhIzZEtPGGR0QzpYw
   （來源：https://b23.tv/ozre4v2）

------------------------------------------------------------------------
二、 檔案配置
------------------------------------------------------------------------
1. 請確保下載成功的模型資料夾名字為「weights」。
2. 把本資料夾的所有檔案和「weights」一同放入你下載的整合包資料夾中。

------------------------------------------------------------------------
三、 啟動服務
------------------------------------------------------------------------
【Windows 用戶】
進入整合包資料夾，雙擊「win_run_tts.bat」即可啟動。

【Mac 用戶】
1. 進入整合包資料夾，打開 Terminal。
2. 使用 cd 指令切換至你的當前路徑。
3. 輸入「chmod +x 」（注意 +x 後面有一個空格）。
4. 把「mac_run_tts.command」拖入 Terminal 中，並按下 Enter（此為第一次使用）。
5. 以後只需要進入整合包資料夾，雙擊「mac_run_tts.command」即可啟動。

------------------------------------------------------------------------
四、 網路設定與部署
------------------------------------------------------------------------
1. 請將「9880」端口穿透至你已設定的網址。
2. 複製該穿透網址，並打開「Vioast/static/main.js」檔案。
3. 找到「https://tts-api.hiruynk.com」字串替換為你的網址並儲存。
4. 重新載入「localhost:8000」，嘗試在 Live2D 模式中對話，本地的 TTS-API 就會啟動。
